**SIPENCA - Sistem Informasi Pelestarian dan Eksplorasi Cagar Budaya Sleman**
<hr>
Deskripsi
SIPENCA adalah platform inovatif yang bertujuan memfasilitasi pelestarian dan eksplorasi cagar budaya di Kabupaten Sleman. Dengan memanfaatkan data dari Geoportal Sleman dan Dinas Kebudayaan, SIPENCA menyediakan akses mudah dan interaktif ke informasi seputar kekayaan budaya di wilayah tersebut. Menggunakan teknologi Web GIS, platform ini menyajikan data secara visual untuk mendukung manajemen cagar budaya secara efisien. SIPENCA menyajikan peta interaktif yang memungkinkan pengguna menjelajahi lokasi cagar budaya di Kabupaten Sleman.
<hr>



**Komponen Pembangun Produk**

1. HTML (Hypertext Markup Language): Digunakan untuk struktur dasar halaman web.
2. CSS (Cascading Style Sheets): Digunakan untuk mengatur tata letak dan tampilan halaman web.
3. Bootstrap: Framework CSS yang digunakan untuk membangun tampilan responsif dan menarik.
4. PHP (Hypertext Preprocessor): Digunakan sebagai bahasa pemrograman server-side untuk pengembangan aplikasi web dinamis.
5. GeoServer: Digunakan sebagai server penyedia layanan peta geospasial.
6. Web GIS: Menggunakan teknologi web GIS untuk menyajikan data geografis secara interaktif, seperti peta interaktif.
7. JavaScript: Digunakan untuk memberikan interaktivitas pada halaman web.

**Sumber Data**

1. Data Geoportal Sleman : https://geoportal.slemankab.go.id/ (kebutuhan data titik)
2. Data Dinas Kebudayaan : https://jogjacagar.jogjaprov.go.id/ (kebutuhan dokumentasi data)
3. Data Badan Informasi Geospasial : https://tanahair.indonesia.go.id/portal-web (kebutuhan data poligon)
